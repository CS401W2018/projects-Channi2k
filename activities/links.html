Explain the difference between an absolute link and a relative link. Why would you use relative links?

An absolute link contains the full web address, meanwhile; relative links primarily to link to pages within the same website because they are shorter, easier to manage, and automatically adjust if the website's domain changes.

List the 3 rules with relative links

  1. Relative URLs are only for files on the same server.
  2. If a file is in the same directory as the HTML file containing the link, we just give its name.
  3. If a file is in a sibling subdirectory, we can just give the path starting from the sibling.
