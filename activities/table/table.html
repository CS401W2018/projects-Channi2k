Questions

  1. Why don't we use the border attribute?

    2. Nowadays it's uncommon to style elements using html attributes. It's better to apply styling through CSS.

  2. What is the easiest way to color every other row a different color?

    3. The easiest way to color every other row in an HTML table is to use CSS's :nth-child(odd) and :nth-child(even) pseudo-classes to target the odd and even rows respectively, and then apply different background colors.
