Explain the difference between inline, embedded, and external CSS.

Styles that are utilized within the HTML tag itself are known as inline styles. A page's header has embedded styles that are applied to the entire page. Each page of the website can link to an external style sheet, which is a distinct text file, to obtain its instructions.

Explain why you should only use external CSS.

Because external styles are contained in a different file, the HTML file size is decreased and cached for quicker page loading.

What is a selector used for?

CSS selectors are used to "find" (or select) the HTML elements you want to style.
