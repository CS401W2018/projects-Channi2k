List out the different ways to select an item.

  1. Simple selectors (select elements based on name, id, class)
  2. Combinator selectors (select elements based on a specific relationship between them)
  3. Pseudo-class selectors (select elements based on a certain state)
  4. Pseudo-elements selectors (select and style a part of an element)

What is the difference between an ID and a Class?

ID is specific to a single element, while classes can be assigned to multiple elements

What is the difference between a SPAN tag and a DIV tag?

div is a block-level element creating a new line and used for structuring larger sections, span is an inline element for styling small segments of text without altering the document flow

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

div, header, and footer all works the sane, but what differentiate them is the name

What are the different ways to select a color?

In CSS, you can select a color using several methods, including: color names (keywords), hexadecimal codes, RGB (Red, Green, Blue) values, RGBA (Red, Green, Blue, Alpha) values, HSL (Hue, Saturation, Lightness) values, and HSLA (Hue, Saturation, Lightness, Alpha) values
