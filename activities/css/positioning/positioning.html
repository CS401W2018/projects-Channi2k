1. What is normal flow in HTML?

Normal Flow is the way that block and inline elements are displayed on a page before any changes are made to their layout

2. What does float do? Why do you need to clear the float?

when an element floats, it is removed from the normal document flow, meaning subsequent elements might appear above it instead of below, so "clear" forces the next element to start below any floated elements, creating a proper layout.

3. What do absolute, fixed, and relative do?

In CSS, absolute, fixed, and relative positioning are ways to define how elements appear on a web page.
