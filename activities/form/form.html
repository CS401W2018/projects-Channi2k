Questions:

  1. The form tag has an "action" attribute. What does it do?

    2. The action attribute defines the action to be performed when the form is submitted.

  2. The form tag has a "method" attribute. What does it do?

    3. The method attribute in an HTML form tag specifies the HTTP method used to send form data to the server, which can be either GET or POST.

  3. What is the difference between a "POST" and a "GET"?

    4. You use GET when you want to retrieve data and use POST when you want to submit data.

  4. What are different ways to style your form?

    5. By adding Width, Padding, Margin, Border, etc. in Input Field

  5. Why does the professor like to use flex?

    6. Flex makes the alignment looks good. One of the main advantages of flexbox is the ability to fill extra space without the need to use Javascript.
